O que é Cloud Server?

29/01/2026 2 min leitura Compartilhar
Publicidade Google

Cloud server é um serviço de computação em nuvem que permite a ativação facilitada de servidores virtuais para você migrar ou rodar aplicações na nuvem com alta confiabilidade.

Através de um painel de autosserviço, criado para simplificar o gerenciamento e a análise do consumo de recursos, você ativa ou remove servidores, gera imagens e pontos de restauração, configura backups, acessa o terminal e os dados de monitoramento.

Conheça os principais componentes:

  • Software de gerenciamento em nuvem, composto por programas como VMware, Nutanix e Proxmox;
  • Software de virtualização, como o hypervisor KVM, com recursos de alta disponibilidade de recursos, alocação simplificada, troca de servidores sem parada, switch virtual, controle de memória, processamento, espaço em disco, IOPS, entre outros;
  • Alocação dinâmica dos cloud servers no cluster de hospedeiros, para garantir o pleno acesso à capacidade computacional contratada;
  • Servidores hospedeiros, com alta capacidade de processamento e memória e baixo consumo de energia;
  • Camada de storage SSD virtualizada (Ceph), com redundância total e expansão transparente de capacidade;
  • Hardware de rede e conexões redundantes.

O software de virtualização realiza a alocação dos recursos computacionais dos hospdeiros de forma exclusiva e isolada para os cloud servers.

Em cada hospedeiro físico há recursos computacionais disponíveis (CPU, memória, disco e rede) usados pelos cloud servers. Caso um hospedeiro apresente falha, um mecanismo de alta disponibilidade entra em ação e reinicia automaticamente os cloud servers em um novo equipamento.

Esta infraestrutura oferece servidores na nuvem com facilidade de expansão e suporte a aplicações que demandem alta capacidade computacional e performance.
